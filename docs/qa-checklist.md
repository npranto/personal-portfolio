# QA Checklist

## Automated

- `npm run validate-content`
- `npm run lint`
- `npm run typecheck`
- `npm run build`
- `npm run check`

## Manual

- Verify homepage section order and spacing in mobile/tablet/desktop.
- Verify project filters + search combinations and empty state behavior.
- Verify project detail pages render all expected content blocks.
- Verify skip-link, focus rings, and keyboard-only navigation flow.
- Verify external links open safely with `noopener noreferrer`.
- Verify metadata, sitemap, and robots routes in production build.
